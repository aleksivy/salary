Документ «Оценка кандидата»

Документ «Оценка кандидата» регистрирует смену статуса кандидата на работу.

В документе для каждого кандидата указывается решение, дата его принятия и, если при подборе персонала принято ранжировать кандидатов, указывается оценка кандидата.

При работе с инструментом менеджера по набору кадров документ автоматически создается при смене статуса кандидата.

Автоматическое заполнение на основании других документов. Документ может быть введен на основании документа «Регистрация кандидатов», при этом в табличную часть будет перенесен список кандидатов.

Ввод на основании. На основании документа «Оценка кандидата» можно провести прием на работу кандидатов, по которым в качестве решения указан статус «предложили работу».
