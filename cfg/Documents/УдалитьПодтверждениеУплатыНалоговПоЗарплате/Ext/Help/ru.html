Документ «Подтверждение уплаты налогов по зарплате»

Документ предназначен для подготовки данных для формирования отчетности. Документ поможет правильно отразить сумму начислений по факту выплаты заработной платы в разрезе работников, а также отразить в разрезе работников факт перечисления взносов в фонды социального страхования и НДФЛ.

 

Документ необходимо заполнить перед формированием регламентированной отчетности, а именно отчетов:
РегламентированныйОтчетФорма1ДФ

РегламентированныйОтчетФССНесчастныйСлучай

РегламентированныйОтчетПенсионныйФонд

 

Документ заполняется один раз в месяц, после начисления заработной платы и осуществления всех платежей за отчетный период.
Важен порядок заполнения реквизитов. Сначала заполняется дата и отчетный месяц, затем производится заполнение с помощью кнопки "Заполнить" (панели действий формы).
После того как произведено полное заполнение, значение табличных частей можно отредактировать вручную. При полном заполнении табличные части расчитываются в следующем порядке:
- Работники организации
- Начисления
- Взносы
- НДФЛ
 
Причем табличные части "Начисления", "Взносы" и "НДФЛ" формируются только по работникам фигурирующим в табличной части "Работники организации".
Для подтверждения уплаты налогов и взносов анализируются только налоговые периоды, которые фигурировали как периоды взаиморасчетов в табличной части "Начисления".
То есть при самостоятельном заполнении табличных частей, рекомендуется придерживаться вышеприведенного порядка.
 
Табличная часть "Начисления"
Предназначена для формирования суммы "Начислено по факту выплаты", значение других колонок - информативное. при заполнении и проведении документа анализируется начисленная зарплата, остатки взаиморасчетов работников и выплаты работникам на дату документа.
 
Табличная часть "Взносы"

Для формирования регламентированной отчетности по взносам в фонды социального страхования используется регистр "Взаиморасчеты по Взносам в фонды".
В табличной части отражается факт уплаты взносов в социальные фонды на фонд оплаты труда в разрезе каждого работника, а также факт перечисления предприятием взносов работников. Подтвердить уплату можно только по взносам, которые были начислены. Начисление осуществляется документами "Начисление зарплаты работникам организаций", "Расчет при увольнении работника организации". Если выплаты заполненные вручную превышают начисленные суммы, то такое заполнение будет считаться некорректным и документ проведен не будет.
Так же проверяется корректность общей сумма перечисления взносов. Перечислить взносы можно с помощью платежных документов и на момент формирования подтверждения эти суммы должны быть заполнены.
 
Табличная часть "НДФЛ"
Для формирования регламентированной отчетности по налогу на доходы физических лиц используется регистр "Взаиморасчеты по НДФЛ".
В табличной части отражается факт уплаты взносов в социальные фонды на фонд оплаты труда в разрезе каждого работника, а также факт перечисления предприятием взносов работников. Подтвердить уплату можно только по взносам, которые были начислены. Начисление осуществляется документами "Начисление зарплаты работникам организаций", "Расчет при увольнении работника организации", "Перерасчет НДФЛ". Если выплаты заполненные вручную превышают начисленные суммы, то такое заполнение будет считаться некорректным и документ проведен не будет.
Так же проверяется корректность общей сумма налога, которая перечислена.
