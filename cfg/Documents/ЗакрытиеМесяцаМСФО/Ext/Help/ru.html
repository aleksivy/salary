Закрытие месяца МСФО

 

 

Документ, который проводит большую часть регламентных операций по международному учету.

В документе «Закрытие месяца (МСФО)» можно выбрать подсистему, по которой ведется учет.

Документ состоит из перечня статей регламентных операций бухгалтерского и налогового учета, установленных в порядке их закрытия. Каждая регламентная операция для каждой организации должна быть произведена только один раз за каждый месяц.

 

Начисление амортизации основных средств (МСФО)

 

При проведении документа с установленным флажком «Начисление амортизации ОС МСФО» будет рассчитана амортизация тех объектов основных средств, для которых стоимость погашается посредством начисления амортизации.

 

Начисление амортизации нематериальных активов (МСФО)

 

При проведении документа с установленным флажком «Начисление амортизации НМА МСФО» будет рассчитана амортизация  объектов по указанным видам учета.

 

Переоценка валютных средств (МСФО)

 

Статья предназначена для переоценки сумм в валюте регламентированного учета по счетам денежных средств и взаиморасчетов с контрагентами и подотчетными лицами. Не делается переоценка валютных средств по тем счетам, которые включены в «Счета с особым порядком переоценки МСФО». 

При переоценке валютных средств считается неизменным остаток в валюте (в валюте денежных средств, валюте учета долга подотчетного лица, валюте взаиморасчетов с контрагентом), а остатки в валюте ведения регламентированного учета рассчитываются по курсу, указанному в справочнике валют на момент проведения переоценки.

Если текущий остаток отличается от рассчитанного остатка денежных средств, то он корректируется, при этом величина корректировки является курсовой разницей.

Движения, которые фиксируются документом «Закрытие месяца МСФО» по переоценке валютных средств, и будут являться курсовыми разницами.

 

Закрытие затратных счетов

 

Для возможности задания параметров закрытия затратных счетов в конце месяца существует регистр сведений «Закрытие затратных счетов МСФО». В качестве параметров закрытия указываются

1.                  затратный счет, который необходимо закрывать. При этом существует возможность задания значений субконто для настройки условий для закрытия счетов;

2.                  счет, на который будет выполнено закрытие. Для счета возможно задание предопределенных значений субконто, которые будут подставляться в проводки при выполнении закрытия сумм;

3.                  тип сумм для закрытия (остатки или обороты). Использование данного параметра вызвано наличием ведения учета по затратным счетам только по оборотам (без хранения в системе остатков). Для таких счетов подразумевается, что они обязательно должны быть закрыты в конце отчетного периода;

База распределения сумм определяет, в какой пропорции суммы с затратных счетов будут перенесены на счет для закрытия.
