Загрузка входящих первичных документов из "FREDO ДокМен"

Обработка предназначена для пакетной загрузки входящих первичных докуементов (Счетов, Накладных, Актов) , полученных от контрагентов в электронном виде в формате XML через систему "FREDO ДокМен".

Порядок работы

  1. Таблица полученных документов в "FREDO ДокМен" заполняется при открытии формы или при нажатии на кнопку "Обновить список". В таблицу попадают документы, выписанные контрагентами за указанный период по выбранной организации. Возможно также установить отбор по определенному контрагенту.
  2. После получения списка входящих документов из "FREDO ДокМен" система автоматически определяет контрагента из информацинной базы по коду ЕГРПО(ДРФО).
  3. Если входящий документ уже регистрировался в информационной базе 1С, ссылка на такой документ будет показана в таблице. Также будет произведен анализ найденного документа из информационной базы 1С на соответствие ключевых полей. Информация о найденных расхождениях будет показана в таблице.
  4. После загрузки списка документов из "FREDO ДокМен" в таблице будут установлены флажки "загружать ?" для тех строк для которых не были обнаружены соответствующие документы в системе 1С либо в таких документах были обнаружены расхождения в ключевых полях. Если не удалось в информационной базе найти контрагента с соответствующими данным "FREDO ДокМен" кодами флажок загрузки также будет снят.
  5. Для строк таблицы, в которых не указан контрагент необходимо его указать вручную.
  6. Загрузка входящих первичных документов просихожит при нажатии на кнопку "Загрузить документы".
