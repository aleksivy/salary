Настройка выгрузки данных в 1С:Бухгалтерию 8 для Украины

В полях «Период с» – «по» укажите период, за который будет производиться выгрузка документов. В поле «Файл данных» укажите имя внешнего файла, в который будет производиться выгрузка данных. Кнопка «Выполнить» позволяет выгрузить данные в указанный в обработке файл.

Выбор редакции. Необходимо выбрать редакцию конфигурации "1С: Бухгалтерия для Украины", в которую осуществляется выгрузка. Выгрузка в редакцию 1.2 возможно только для версии "1С: Бухгалтерии для Украины" не ниже 1.2.3. Выгрузку в редакцию 1.1 следует осуществлять только для документов до 01.04.2011 г.!

Объем выгружаемых данных зависит от установок, указанных в поле «Выгружаемые объекты».

Выгрузка данных по отражению зарплаты в регламентированном учете. Должен быть взведен флажок «Отражение зарплаты в регл учете»; при этом будут выгружаться документы "Отражение зарплаты в регл учете", "Списание депонентов в доходы организаций" и "Начисление коммунального налога" дата которых попадает в указанный период.

Выгрузка кассовых документов. Должен быть взведен флажок «Кассовые документы»; при этом будут выгружаться следующие документы: «Зарплата к выплате организации», «Приходный кассовый ордер», «Расходный кассовый ордер», «Платежное поручение исходящее».

Выгрузка поступления денежных средств от ФСС. Должен быть взведен флажок «Поступления средств от ФСС»; при этом будут выгружаться следующие документы: «Платежное поручение входящее».
