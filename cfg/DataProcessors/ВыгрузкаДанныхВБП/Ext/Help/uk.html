Выгрузка данных в 1С:Бухгалтерию 8 для Украины

В программу «1С:Бухгалтерия 8 для Украины» передаются сведения об отражении в бухгалтерском и налоговом учете сумм начисленной заработной платы и соответствующих налогов и взносов, кассовые и банковские документы по выплате зарплаты.

Общие правила обмена данными

  1. Перед очередным формированием экспортного файла с данными учета начисленной заработной платы (или при внедрении конфигурации «Зарплата и Управление Персоналом для Украины» в эксплуатацию) необходимо в программе «1С:Бухгалтерия 8 для Украины» сформировать файл с описанием объектов аналитического учета обработкой «Выгрузка данных в конфигурацию "Зарплата и управление персоналом для Украины"» программы «1С:Бухгалтерия 8 для Украины» (см. меню Сервис).
  2. Сформированный в программе «1С:Бухгалтерия 8 для Украины» файл с описанием объектов аналитического учета следует загрузить в информационную базу конфигурации «Зарплата и Управление Персоналом для Украины» при помощи универсальной обработки обмена данными. Для этого необходимо:
    · выбрать в меню "Обмен данными" (меню "Предприятие") пункт "Универсальный обмен данными в формате XML";
    · в появившемся диалоге на закладке "Загрузка данных" указать имя файла с описанием объектов аналитического учета;
    · нажать на кнопку "Выполнить".
  3. Если в используемой "бухгалтерской" аналитике произошли какие-либо изменения, тогда необходимо отредактировать способы отражения зарплаты в учете, используемые в конфигурации «Зарплата и Управление Персоналом для Украины» (см. меню "Учет зарплаты", интерфейс "Расчет зарплаты организаций").
  4. Сформировать и провести документы по отражению начисленной зарплаты в регламентированном учете за очередной расчетный месяц.
  5. Выгрузить в файл данные об отражении начисленной зарплаты в регламентированном учете настоящей обработкой. Подробнее см. здесь.
  6. Сформированный в конфигурации «Зарплата и Управление Персоналом для Украины» файл с переносимыми данными следует загрузить в ИБ программы «1С:Бухгалтерия 8 для Украины». Для этого также используется универсальная обработка обмена данными.

Переносимые данные

В программу «1С:Бухгалтерия 8 для Украины» могут быть выгружены следующие документы:

  • документы по отражению зарплаты в учете:
      • "Отражение зарплаты в регл учете";
      • "Списание депонентов в доходы организаций".
  • документы по выплате зарплаты:
      • «Зарплата к выплате организации»;
      • «Расходный кассовый ордер»;
      • «Приходный кассовый ордер»;
      • «Платежное поручение исходящее».
  • документы поступления денежных средств от ФСС.
