Форма «Настройка прав доступа»

В форме настройки прав доступа производится настройка прав доступа пользователей к конкретным элементам «общих» справочников. Доступ к этим справочникам предоставляется пользователям через обязательную роль «Пользователь». Настройка прав доступа для каждого пользователя определяет, с какими элементами «общих» справочников ему можно работать.

Форма настройки прав доступа разделена на две части. В левой части отображается список областей ограничения прав доступа «общих» справочников. Для справочников, для которых определены несколько областей ограничений прав доступа (например, «Контрагенты», «Организации»), доступ к конкретному элементу справочника настраивается отдельно для кадой области.

В правой части формы производится настройка прав доступа для текущего справочника (текущей области ограничения прав доступа).

В колонке «Объект доступа» указывается конкретный элемент справочника, к которому настраивается доступ. Для иерархических справочников можно выбирать группы. Если не выбран конкретный элемент или группа справочника, то настройка будет действовать для всего справочника.

Для каждого элемента справочника можно установить следующие границы доступа:

  • Чтение. Пользователь будет видеть элемент справочника в списке и сможет выбрать его из списка при заполнении реквизитов других объектов.
  • Изменение. Пользователь может открыть элемент справочника и связанную с ним дополнительную информацию, изменить информацию и сохранить эти изменения.
  • Добавление. Пользователь может добавить информацию, связанную с этим элементом справочника, а также добавить новый элемент справочника, если добавление разрешено при настройке доступа ко всему справочнику в целом.
  • Настройка.  Пользователь может изменить настройку доступа к этому элементу для других пользователей программы.

Права доступа назначаются установкой флажков в соответствующих колонках.

В колонке «Вид наследования» устанавливается вид наследования прав доступа для текущего элемента, при этом:

  • Для элементов неиерархического справочника и элементов иерархического справочника с иерархией групп и элементов всегда устанавливается вид наследования «Для текущего элемента».
  • Для групп иерархического справочника с иерархией групп и элементов всегда уснавливается вид наследования «Распространить на подчиненных».
  • Для элементов иерархического справочника с иерархией элементов можно установить вид наследования «Распространить на подчиненных» или «Для текущего элемента».
  • При настройке прав доступа ко всему справочнику в целом (если в колонке «Объект доступа» не выбрано конкретное значение), для справочников с любым видом иерархии всегда устанавливается вид наследования «Распространять на подчиненных».

Права доступа назначаются установкой флажков в соответствующих колонках.

Для элементов или групп с видом наследования «Распространять на подчиненных» установленные им права доступа будут автоматически назначаться для всех дочерних объектов с признаком «Унаследовано от родителя». Если права доступа настроены для всего справочника в целом, то установленные права будут автоматически назначаться всем элементам и группам элементов этого справочника с признаком «Унаследовано от родителя».

Если права доступа для иерархических справочников задаются для элемента или группы с признаком «Только для текущего права» или «Распространять на подчиненных», то для всех родительских групп (иерархия групп и элементов) и для всех родительских элементов (иерархия элементов) будет установлено право доступа на чтение с признаком «Унаследовано от дочернего». Это позволяет пользователю видеть в списке справочника все групп, в которых ему доступен для чтения хотя бы один элемент или группа. У записей с видом наследования «Унаследовано от родителя» или «Унаследовано от дочернего» изменять состояние флажков нельзя. Все изменения для них можно сделать только через родительские или дочерние записи соответственно.
